# UI7 eSports

Professional eSports Tournament Platform built with React, Vite, and Tailwind CSS.

## Features

- 🎮 Tournament Management
- 💰 Paid Scrims
- 👥 Team Management
- 📅 Schedule Management
- 🌙 Dark Mode Support
- 📱 Responsive Design

## Quick Start

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173) to view the app.

### Production Build

```bash
npm run build
```

## Tech Stack

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Hosting**: Vercel

## Project Structure

```
src/
├── components/      # Reusable React components
├── context/         # React Context for state management
├── pages/          # Page components
├── App.jsx         # Main App component
└── main.jsx        # Entry point
```
